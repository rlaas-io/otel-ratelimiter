3) Decision Logic (easy mental model)

for each record in batch:
  if max_batch_size exceeded:
    drop(reason=batch_size_exceeded)
    continue

  decision = evaluate(policy)

  if evaluate error:
    if fail_open: keep
    else: drop(reason=error)
    continue

  if decision.shadow_mode:
    keep and count as shadow
  else if decision.allowed:
    keep and count as allowed
  else:
    drop(reason=rate_limited)
