Behavior guarantees

  • Fail-open mode prevents telemetry loss when evaluation fails.
  • Fail-closed mode enforces strict control for compliance-sensitive paths.
  • Shadow mode validates policy safety before hard enforcement.
  • Admin API exposes runtime state for operations and incident response.
