Production Examples

Minimal deployment

processors:
  ratelimiter:
    policy_file: /etc/otelcol/policies.json
    fail_open: true

Hot reload + admin API

processors:
  ratelimiter:
    policy_file: /etc/otelcol/policies.json
    watch_policies: true
    watch_interval: 15s
    admin_addr: ":8090"

Inline policies (Helm / ConfigMap friendly)

processors:
  ratelimiter:
    fail_open: true
    policies_inline: |
      [
        {
          "id":"helm-logs",
          "name":"Helm logs limiter",
          "signal_type":"log",
          "service":"checkout",
          "algorithm":"token_bucket",
          "rate":300,
          "burst":900,
          "action":"drop"
        }
      ]

Protect against huge batches

processors:
  ratelimiter:
    policy_file: /etc/otelcol/policies.json
    max_batch_size: 10000

Operator quick checks

curl http://localhost:8090/health
curl http://localhost:8090/stats
curl -X POST http://localhost:8090/reload
curl http://localhost:8090/metrics
Repo examples you can start from: example/otel-collector-config.yaml and example/policies.json.
